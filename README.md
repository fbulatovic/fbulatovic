### [FMEA Excellence](https://fmea-excellence.com) — *Lead Frontend Dev*
> AIAG/VDA 2019-compliant PFMEA/DFMEA platform for the automotive quality industry

- Complex risk tables with TanStack Table (inline editing, fuzzy filters, RPN/AP color logic)
- Structural analysis graphs via React Flow
- Analytics dashboard with Recharts (stacked bar + Pareto)
- Real-time autosave, URL-synced navigation, filter persistence

> Job application tracker with a full auth system and a polished dark UI

- NestJS + PostgreSQL backend, JWT + httpOnly refresh cookies
- TanStack Table with inline editing, CV upload, admin panel

- Deployed on Netcup VPS via GitHub Actions CI/CD
